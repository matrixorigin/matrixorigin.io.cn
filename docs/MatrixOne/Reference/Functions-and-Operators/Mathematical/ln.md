# **LN()**


> LN(X) 返回 X 的自然对数。这个函数是 LOG(X) 的同义函数。

## **函数说明**

LN(X) 返回 X 的自然对数。这个函数是 LOG(X) 的同义函数。

## **函数语法**

```
> LN(X)
```

其他请参考 [LOG(X)](log.md)。
