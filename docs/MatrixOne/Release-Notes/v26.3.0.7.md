# **MatrixOne v25.3.0.7 发布报告**  

发布日期：2026 年 2 月 22 日  

MatrixOne 版本：v25.3.0.7  

MatrixOne 3.0.7 引入了一系列改进和错误修复，以提升系统的易用性。以下是主要更新内容：  

## **错误修复**  

- 修复 pipeline 客户端泄漏问题

- 修复在同一个事务中执行 CREATE AS SELECT，并强制检查源表的 SELECT 权限

- 修复 CTAS（Create Table As Select）在后续强制插入时只能在单个 CN 上运行的问题

- 修复使用 ORDER BY FIELD() 函数的 UNION 语句问题

- 修复 CDC（变更数据捕获）截断操作的错误
