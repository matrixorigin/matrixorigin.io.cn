# **MatrixOne v25.3.0.4 发布报告**  

发布日期：2025 年 10 月 15 日  

MatrixOne 版本：v25.3.0.4  

MatrixOne 3.0.4 引入了一系列改进和错误修复，以提升系统的易用性。以下是主要更新内容：  

## **优化**  

- 提升向量索引的性能
- 支持 remap 提示（remap hint）

## **错误修复**  

- 修复 GC 相关的一些错误

- 修复 CDC 状态和 sinker 的错误

- 修复在 `PreInsertFullTextIndex` 中，当列未更新时检查分区并跳过的逻辑错误

- 修复统计信息中十进制负值转换错误的问题

更多错误修复详情，请参阅 [MatrixOne-v25.3.0.4](https://github.com/matrixorigin/matrixone/releases/tag/v3.0.4)。
