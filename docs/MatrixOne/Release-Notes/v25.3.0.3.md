# **MatrixOne v25.3.0.3 发布报告**  

发布日期：2025 年 10 月 15 日  

MatrixOne 版本：v25.3.0.3  

MatrixOne 3.0.3 引入了一系列改进和错误修复，以提升系统的易用性。以下是主要更新内容：  

## **错误修复**  

- 修复删除 ivfflat 索引失败的问题

- 修复 GC 文件泄漏问题

更多错误修复详情，请参阅 [MatrixOne-v25.3.0.3](https://github.com/matrixorigin/matrixone/releases/tag/v3.0.3)。
