# **MatrixOne v25.3.0.8 发布报告**  

发布日期：2026 年 2 月 27 日  

MatrixOne 版本：v25.3.0.8  

MatrixOne 3.0.8 引入了一系列改进和错误修复，以提升系统的易用性。以下是主要更新内容：  

## **错误修复**  

- 修复：在获取独占锁后刷新快照，以防止并发 CLONE 导致的孤儿记录

- 修复：为 INSERT 操作预构建复合唯一键锁列映射的问题
