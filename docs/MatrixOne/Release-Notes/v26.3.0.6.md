# **MatrixOne v25.3.0.6 发布报告**  

发布日期：2026 年 1 月 28 日  

MatrixOne 版本：v25.3.0.6  

MatrixOne 3.0.6 引入了一系列改进和错误修复，以提升系统的易用性。以下是主要更新内容：  

## **优化**  

- Bloom Filter 与多 CN：支持仅使用 Bloom Filter（无需常规过滤器），并在多个 CN 间广播 Bloom Filter，以提升分布式查询性能。

- 备份与导出：为备份增加文件索引，并修复表数据导出问题，使备份/恢复与导出功能更加可靠。

- 向量相关特性：启用 simsimd（MO_CL_SIMSIMD=1），支持在预编译语句中使用向量索引，完善向量相关测试用例以提升性能与稳定性。

- 优化器与执行层：改进 UNION 结果的稳定性，优化过滤下推和 Hint 处理逻辑（包括 CTE 与类型匹配），修正模运算（modulo）场景下 NDV（去重值基数）估计不准确的问题。

- GC 与存储：优化 GC 行为，修复 tmp-fs GC 日志中 nil 条目的处理问题，根据事务和表对 CN segment map 进行作用域划分，以避免因克隆导致的删除跳过问题。

## **错误修复**  

- 修复在数据导入时发生丢包后可能导致的卡死问题。

- 修复自增服务缓存不一致的问题，稳定嵌套 SQL/自增重试路径，并移除多余的自增日志。

- 修复多 CN 扫描统计不一致问题、一个窗口函数兼容性问题，以及修复越界导致的崩溃。

- 修正 x86_64 的默认 MARCH（之前错误地设为 haswell），以避免兼容性和性能问题。
